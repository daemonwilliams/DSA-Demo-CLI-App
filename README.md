# Data Structures & Algorithms CLI App

An interactive Command Line Interface (CLI) application demonstrating core Data Structures and Algorithms in C. This project serves as both a reference implementation and an educational tool.

## The Srructure

- **src/:** Implementation of the Data Structures & Algorithms
- **docs/:** Notes in markdown explaining implementations and concepts

## Features

- **Interactive Demos:**
  - **Hash Table**
  - **Binary Search Tree**
  - **Linked List**
  - **Sorting:**
    - **Quick Sort**
    - **Merge Sort**
- **Integrated Quizzes**

## Installation

Run the following command from the project root (`DSApp/`):

```bash
gcc main.c src/hash.c src/bst.c src/linked_list.c src/sorting.c src/stack.c src/queue.c -o dsapp
```

Run the compiled executable:

```bash
./dsapp
```

Follow the on-screen menu to navigate between different data structure demos.
